# Phishing Email Detector

A machine learning web application that detects phishing emails using NLP.

## Tech Stack

- Python
- Scikit-learn
- Flask
- NLP (TF-IDF)

## Features

- Analyze email text
- Detect phishing attempts
- Web interface for testing

## How to Run

```bash
pip install -r requirements.txt
python train_model.py
python app.py
